# Foodies 🍔

A modern and responsive food ordering application built with React, Redux Toolkit, and Tailwind CSS. The project focuses on delivering a smooth user experience through performant state management, engaging animations, and a scalable front-end architecture.

[View Demo](https://foodies-fastfood.netlify.app/)

**Note:** If you are in Iran, you may need to use a VPN to access the demo link.

![demo](https://github.com/parvin-noori/foodies/blob/main/src/assets/imgs/screenshot/smartmockups_foodies.jpg)

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Global state management with Redux Toolkit
- Interactive animations with Framer Motion
- Reusable and scalable component architecture
- Modern UI built with Tailwind CSS
- Fast development and optimized production builds with Vite

## 🛠️ Tech Stack

- React 18
- Redux Toolkit
- React Redux
- Framer Motion
- Tailwind CSS
- Vite
- React Icons
